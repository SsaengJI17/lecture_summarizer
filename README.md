# Lecture Summarizer

실시간 강의 내용을 받아 적고, 요약과 검색을 통해 수업 내용을 다시 확인할 수 있도록 만든 학습 보조 웹 애플리케이션입니다.

## 주요 기능

- 실시간 강의 음성 받아쓰기
- 구간별 요약 생성
- 전체 요약 생성 설정
- 검색 및 검색 결과 이동
- 실시간 전문 블록 하이라이트
- PIP 모드 지원
- 한국어 중심 사용 흐름
- 초기화 버튼 안전장치

## 프로젝트 목적

강의 내용을 들으면서 동시에 필기하기 어려운 상황을 보완하기 위해 만든 개인용 학습 도구입니다. 수업 중에는 실시간으로 내용을 기록하고, 수업 후에는 요약과 검색 기능을 통해 필요한 부분을 빠르게 다시 찾는 것을 목표로 합니다.

## 사용 예시

1. 수업 또는 강의 중 앱을 실행합니다.
2. 음성 인식을 통해 강의 내용을 실시간으로 기록합니다.
3. 필요한 경우 구간별 요약 또는 전체 요약을 생성합니다.
4. 검색 기능으로 특정 키워드가 나온 위치를 다시 확인합니다.
5. PIP 모드로 다른 작업과 함께 확인합니다.

## 개발 과정에서 중점적으로 본 부분

- 실시간 갱신 중에도 검색 결과 이동이 자연스럽게 동작하는지
- 요약 블록과 검색 하이라이트가 겹칠 때 가독성이 유지되는지
- 사용자가 원하지 않는 자동 요약이 실행되지 않도록 설정 상태가 잘 반영되는지
- 한국어 강의 환경에서 언어 인식이 안정적으로 유지되는지
- PIP 모드에서도 주요 조작이 가능한지

## 향후 개선 사항

- 실행 방법 및 설치 과정 정리
- 사용 화면 스크린샷 추가
- 음성 인식 언어 고정 옵션 개선
- 요약 생성 조건 세분화
- 검색 하이라이트 레이어 안정화
- 수업별 기록 저장 및 불러오기 기능
- 배포용 패키징 정리

## 저장소 상태

현재는 개인 학습 및 실험 목적으로 개발 중인 프로젝트입니다. 기능 안정화와 문서화를 계속 보완할 예정입니다.
